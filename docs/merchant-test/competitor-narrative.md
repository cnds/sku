# Competitor Narrative

SKU Lens should not compete as a broader analytics dashboard or AI audit tool.

## Positioning Test

Ask four questions:

1. Does the product tell the merchant which SKU matters today?
2. Does the recommendation include behavior evidence?
3. Does it give one first action instead of a suggestion list?
4. Does the merchant need to inspect more charts before acting?

SKU Lens wins when the answer is clear: one board, three decisions, evidence, and a first fix.

## Against Dashboards

Dashboards are useful for exploration, but they shift judgment back to the merchant. SKU Lens should keep scores internal and present a small set of action cards.

## Against AI Audit Tools

Generic audits can produce long checklists. SKU Lens should stay grounded in observed product behavior and avoid advice that is not tied to the funnel or PDP evidence.

## Messaging Boundary

Do not claim automatic optimization or guaranteed revenue lift. The current promise is prioritization and evidence-backed first actions.
