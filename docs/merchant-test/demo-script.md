# Merchant Demo Script

## Setup

Use `sku-dev-uaop8pff.myshopify.com` or a real test shop with the theme app embed enabled. Start on `/onboarding?shop=<shop>&window=24h`, then open the board.

## Walkthrough

1. Confirm install status, public token, ingest endpoint, App Embed deep link, and raw-event status.
2. Explain that first raw events can arrive before SKU Lens has enough traffic for confident cards.
3. Open the board and frame the product as `2 Leakers + 1 Hidden Winner`.
4. For each card, read the reason, evidence, suspected friction, and first fix.
5. Use one feedback button: `I will try this`, `Not useful`, `Already fixed`, or `Remind me later`.
6. Open the internal card review only for team review, not merchant presentation, and inspect raw events, aggregate evidence, derived signal, AI summary, and final copy.

## Boundary

This script verifies the repo flow. A real merchant pilot still needs live install, real traffic, and post-demo feedback.
