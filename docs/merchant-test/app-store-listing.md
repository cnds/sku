# App Store Listing Draft

## Name

SKU Lens

## Tagline

Three product decisions for today: two Leakers to fix and one Hidden Winner to scale.

## Short Description

SKU Lens turns Shopify storefront behavior into a daily product decision board. Instead of asking merchants to inspect dashboards, it highlights products with shopper drop-off evidence and one first action to try.

## Key Points

- Shows up to three priority cards per window.
- Separates Winners from Leakers using behavior evidence, not sales rank alone.
- Explains the signal, suspected friction, and first fix.
- Shows tracking health so merchants know when data is missing.
- Collects lightweight recommendation feedback before adding heavier workflows.

## Review Boundary

Avoid precise revenue-lift claims until order attribution, recommendation history, and outcome verification are stronger.
