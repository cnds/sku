# Merchant-Test Materials

These files support the repository-verified merchant-test readiness pass. They are not evidence that a live merchant pilot or Shopify App Store review has already completed.

- `demo-script.md`: walkthrough for a merchant demo.
- `app-store-listing.md`: concise App Store draft copy.
- `landing-copy.md`: landing-page draft copy.
- `competitor-narrative.md`: positioning against analytics dashboards and audit tools.
- `theme-validation-checklist.md`: storefront tracker validation checklist for common Shopify themes.
